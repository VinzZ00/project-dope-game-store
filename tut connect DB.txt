cara connect database

1. install xampp

2. run mysql dan apache

3. di project java kamu klik kanan build path, configure build path, add external jars

4. di pop up nya mu find mysql-connector-java-5.1.49.jar di DoPeGameStore\mysql-connector-java-5.1.49

5. klik open

6. klik apply and close

7. kamu buat database di localhost/phpmyadmin

8. kamu import sql yang diberikan.
